# 数据结构与算法学习笔记

> "Bad programmers worry about the code. Good programmers worry about data structures and their relationships." — Linus Torvalds

欢迎来到我的数据结构与算法学习仓库。这里记录了我在学习过程中对各种核心数据结构的理解、实现以及相关的算法练习。我创建这个仓库的目的是为了巩固理论知识，并通过动手实践来加深理解。

## 🛤️ 学习路径

这个仓库的内容大致按照以下顺序组织，这也是我推荐的学习路径：

1.  **基础数据结构**
    -   [ ] 数组 (Array)
    -   [ ] 链表 (Linked List)
    -   [ ] 栈 (Stack)
    -   [ ] 队列 (Queue)
    -   [ ] 哈希表 (Hash Table)
2.  **树形结构**
    -   [ ] 二叉树 (Binary Tree)
    -   [ ] 二叉搜索树 (Binary Search Tree, BST)
    -   [ ] 平衡二叉树 (AVL Tree)
    -   [ ] 红黑树 (Red-Black Tree)
    -   [ ] 堆 (Heap)
    -   [ ] 字典树 (Trie)
3.  **图结构**
    -   [ ] 图的表示 (邻接矩阵、邻接表)
    -   [ ] 图的遍历 (DFS、BFS)
    -   [ ] 最短路径算法 (Dijkstra, Floyd)
    -   [ ] 最小生成树 (Prim, Kruskal)
4.  **高级数据结构与算法**
    -   [ ] 并查集 (Union-Find)
    -   [ ] 跳表 (Skip List)
    -   [ ] 布隆过滤器 (Bloom Filter)
